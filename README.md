# MuhammadDzakiNurfadhilp_WebDev_Seeker
Eco-Track
Tentang Project
Kalkulator emisi karbon harian interaktif untuk menghitung jejak karbon dari sektor transportasi dan penggunaan energi rumah tangga.

Masalah Iklim yang Ingin Disoroti
Fokus pada isu Efek Rumah Kaca. Aplikasi ini membantu pengguna menyadari bahwa aktivitas harian kecil berkontribusi pada akumulasi gas CO₂ yang memicu pemanasan global.

Jalur Spesialisasi yang Dipilih
[x] A1. Real-time UI Feedback
[x] A2. Interactive Tips (Action Plan)
[ ] A3. Dynamic Result Display
[x] B1. Static File Serving
[ ] B2. The Carbon API
[ ] B3. Smart Validation

Cara Menjalankan Project
Ekstrak file dan buka terminal di folder project.
Jalankan npm install untuk mengunduh dependencies.
Jalankan server dengan perintah node app.js.
Akses http://localhost:3000 di browser.

Tantangan yang Dihadapi
Validasi Input: Mengatasi angka negatif menggunakan Math.abs() agar hasil perhitungan selalu absolut dan akurat.
Manipulasi String: Menggunakan Template Literals (${}) untuk menyisipkan variabel ke dalam teks motivasi secara lebih efisien dan rapi.
Prioritas Fitur: Memutuskan untuk memaksimalkan fitur A1 dan A2 terlebih dahulu untuk menjamin kualitas interaksi pengguna sebelum beralih ke grafik dinamis.

Proyek ini disusun untuk memenuhi tugas seleksi Web Development ITC UPNVYK.
